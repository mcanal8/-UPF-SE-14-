TASCA 2: Altres requeriments de domini

  • El sistema verifica el format d'un work i si no es adequat el layout designer el rectifica.
  • Un work pujat per un autor es verificat per l'editor abans de ser publicat.
  • L'editor decideix si un work forma part d'una col·leccio.
  • L'editor li dona el OK a un work, decideix si va a una col·leccio, li afegeix cobertes i el posa a la venta.
  • Quan l'usuari paga el llibre, pot descarregar-lo complet sense marques d'aigua.
  • L'usuari ha de registrar-se a la web amb un usuari i una password per accedir-hi.
  • L'editor i l'autor poden eliminar works de la web.
